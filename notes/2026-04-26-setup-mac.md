# Setup local no Mac

> Sessão de 26/04/2026.

## Contexto

Cliente clonou o repositório `Vitrine-Virtual` no Mac e quer rodar localmente.
README atual ainda assume Windows/PowerShell — precisa virar cross-platform.

## Checklist Mac

1. **Pré-requisitos** — Node 20+, pnpm 9+, git
   - Recomendado: instalar Node via `nvm` ou Homebrew
   - pnpm: `corepack enable` (já vem com Node) ou `brew install pnpm`
2. **Instalar deps:** `pnpm install`
3. **`.env.local`:** `cp .env.example .env.local` e preencher
4. **Keys do Supabase:** `https://supabase.com/dashboard/project/tfylrbxajzmhsdnynpbx/settings/api`
   - `NEXT_PUBLIC_SUPABASE_URL` = `https://tfylrbxajzmhsdnynpbx.supabase.co`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY` = anon (jwt)
   - `SUPABASE_SERVICE_ROLE_KEY` = service_role (secret)
5. **`IP_HASH_SALT`:** `openssl rand -hex 32`
6. **`SUPER_ADMIN_EMAILS`:** já default no .env.example (`francisco.efjr@gmail.com`)
7. **Criar super-admin no Supabase:**
   - Auth → Users → Add user (com Auto Confirm)
   - SQL: `update profiles set role = 'super_admin' where id = (select id from auth.users where email = ...)`
8. **Rodar:** `pnpm dev`
